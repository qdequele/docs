## Securing your project

This guide explains how to secure your Meilisearch project by managing your master key and safely sending requests to the Meilisearch API using API keys.

## Creating the master key

The master key is essential for securing your Meilisearch project. When using Meilisearch Cloud, a master key is automatically generated for each project, ensuring security by default.

You can view your master key in your project overview:

  <img 
    src="https://raw.githubusercontent.com/meilisearch/documentation/cloud-master-api-keys/assets/images/security/01-master-api-keys.png" 
    alt="API keys interface showing Master key, Default Search API Key, and Default Admin API Key" 
  />
</Frame>

## Obtaining API keys

When your project is protected, Meilisearch automatically generates two API keys:
- `Default Search API Key`
- `Default Admin API Key`

These API keys serve as authorization tokens for secure communication with the Meilisearch API.

### Accessing API keys in Meilisearch Cloud 

Find your API keys in the same section as your master key:
